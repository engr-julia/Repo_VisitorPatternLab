# 🛋️ Furniture Shipping Cost Calculator

> “Calculating shipping has never been this elegant — each furniture knows its visitor!”  

A Java project demonstrating the **Visitor Design Pattern** to calculate shipping costs for different furniture types like **Chairs, Tables, and Sofas**. The shipping logic is completely decoupled from furniture classes, making it **easy to extend** with new furniture types or calculation strategies.  

---

## ✨ Features
- Calculate shipping costs for multiple furniture types  
- Each furniture type can have its **own unique shipping logic**  
- Adding new furniture types or new operations (like taxes) requires **no changes to existing classes**  
- Clean, maintainable Java code with **one class per file**  
- UML diagram included for easy visualization
